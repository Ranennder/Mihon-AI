# Contributing

Thanks for your interest in contributing to Mihon AI.

## Code contributions

Pull requests are welcome.

If you plan to work on a bigger change, open or comment on an issue first so the work does not overlap.

## Prerequisites

- Basic Android development knowledge
- Kotlin
- Android Studio
- A device or emulator for testing

## Notes

- Keep branding and release metadata aligned with `Mihon AI`
- Keep public links pointed at this repository, not upstream services
- If a change affects reader AI behavior, include a short test note in the PR
